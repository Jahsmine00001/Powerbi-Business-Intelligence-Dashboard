# Business Intelligence Dashboard – Sales, Profit & Customer Analytics

## Overview
This project demonstrates an end-to-end Business Intelligence solution built on a Gold-layer data model. The dashboard is designed to support executive decision-making across sales performance, profitability risk, and customer behavior.

## Data Model
- fact_sales (transaction-level grain)
- dim_customers
- dim_products
- dim_date

Star schema optimized for analytical queries and time intelligence.

## Dashboard Pages
### Page 1 – Sales Performance
- Sales trends, YoY growth, volatility
- KPI-led design with supporting momentum visuals

### Page 2 – Profitability & Risk
- Profit stability, category dominance, volatility analysis
- Focus on sustainability, not just totals

### Page 3 – Customer Analytics
- Retention, repeat behavior, concentration risk
- Identification of revenue dependency patterns

## Key BI Concepts Applied
- Defensive DAX
- Correct time intelligence
- Metric validation against trends
- Executive-focused visual design

## Tools
- Power BI
- DAX
- SQL Server
- Git

## Notes
This dashboard prioritizes metric correctness and interpretability over visual density.
